# SpaceTimePractise
## 功能
#### 1. 轮播图
#### 2. 转场动画和一些普通动画(自认惨不忍睹)
#### 3. 下载相关:自定义进度条,断点下载
#### 4. 设置本地相册图片为新头像,及相关判断、存储
#### 5. 用户信息界面,为以后加上微信和 QQ 登陆作准备
#### 6. 显示当前地区天气情况(上海气象局接口),定位
#### 7. 网易新闻,上拉刷新、下拉加载
#### 8. FMDB 存储离线数据,增加清除缓存
