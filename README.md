# SpaceTimePractise
## 功能
#### 1. 轮播图
#### 2. 转场动画和一些普通动画(自认惨不忍睹)
#### 3. 下载相关:自定义进度条,断点下载
#### 4. 设置本地相册图片为新头像,及相关判断、存储
#### 5. 用户信息界面,微信和 QQ 登陆及登陆过期(默认一天)
#### 6. 分享(微信和 QQ )
#### 7. 显示当前地区天气情况(上海气象局接口),定位
#### 8. 网易新闻,刷新功能
#### 9. FMDB 存储离线数据,增加清除缓存
#### 10. 视频列表,可小屏幕播放,刷新功能
